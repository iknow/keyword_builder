# KeywordBuilder

KeywordBuilder is a simple wrapper to create a builder-style DSL for any method
taking keyword arguments.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'keyword_builder'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install keyword_builder
